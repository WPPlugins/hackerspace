Add custom post types useful for hackerspaces and expose information trough the Space API.


== Description ==
The plugin a Projects custom post type useful for Hackerspaces. This custom post type is available to the new \"hacker\" user role. The plugin also implement the Space Api to display informations about the Hackerspace. The open/close information is derived from the Opening Hours plugin.

Main limitations: no membership support, no events support.


== Installation ==
= Prerequistes =
Opening Hours WordPress plugin is required for open/close status.

= Install =
1. Download the Zip archive
2. Unzip the archive
3. Upload the plugin folder to the /wp-content/plugins/ directory or by upload in the web interface
4. In your Admin Panel go to Plugins and activate the hackerspace plugin
5. Edit the plugin settings in the Setting/Hackersapce admin panel


== Frequently Asked Questions ==

= How to make permalinks works with the custom post types =
After plugin activation, unset, and set up again the permalinks.
